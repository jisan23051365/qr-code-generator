# Advanced QR Studio

## Overview
Advanced QR Studio is a modern web-based QR code generator that allows users to quickly create customizable QR codes from text or URLs. It includes features such as color customization, size control, QR history tracking, and downloadable QR images.

## Features
- Generate QR codes instantly
- Customize QR size and colors
- Download generated QR codes
- Dark / Light mode toggle
- Local history storage
- Search and export QR history

## Technologies
- HTML
- CSS
- JavaScript
- QR Code API
- LocalStorage
